## ⚠️ සුරක්ෂිතයි - গুরුত්වපූර්ණ

### Firebase Credentials ගිතුබ් ගබඩාවයින් ඉවත් කරන්න

**🔐 ගිතුබ් ගබඩාවයට Firebase ගිතුබු අපලෝপ් කිරීම අවිරක්ත!**

Firebase API Keys, Project IDs සහ අනෙකුත් සිකිරීම සඳහා:

1. **`.env.local` ගොනුව සෑදුන්න** (වර්ජනයෙන් එක්ස්ට්)
   ```
   VITE_FIREBASE_API_KEY=your_key_here
   VITE_FIREBASE_PROJECT_ID=your_project_here
   ```

2. **`.gitignore` එකතු කරන්න**
   ```
   .env.local
   .env*.local
   ```

3. **පරිසරය විචල්‍ය යොදා ගන්න**
   - `firebase-config.js` දැනට `import.meta.env` භාවිතා කරයි
